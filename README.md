# (TeamProgramming) MakeChart

## Requirements:
* Python 3.X
* matplotlib :
  ```
  console >> python -m pip install matplotlib
  ```
	


## Usage:
* Enter project directory.
* Type :
  ```
  console >> python make_chart.py filename
  ```

#### Example with available file :
 * ```
 	console >> python make_chart.py EEPROM
   ```
